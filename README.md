## Running the Application

### Using Maven

```bash
cd backend
mvn spring-boot:run
```

```bash
cd frontend
npm install
ng serve
```

### Using IDE

Run the main application class directly from your IDE (IntelliJ IDEA, Eclipse, VS Code)

The API will be available at: `http://localhost:8080`
